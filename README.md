# BMLD Studierenden App
Diese App wurde erstellt, um BMLD Studierenden den Studienalltag zu erleichtern. Uns ist es wichtig, alle relevanten Infos an einem Fleck zu haben, um Ordnung, Übersicht und Ruhe zu schaffen. Ausserdem beinhaltet die App einen Hydrierungs-Check-in, Lernhilfe,  Kalender, Studenplan, eine To-Do-Liste und einen Notenrechner.
