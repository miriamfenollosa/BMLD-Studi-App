# BMLD Studierenden App
Diese App wurde erstellt, um BMLD Studierenden den Studienalltag zu erleichtern. Uns ist es wichtig, alle relevanten Informationen an einem Ort zu haben, um Ordnung, Übersicht und Ruhe zu schaffen. Unsere beinhaltet die App eine To-Do-Liste, einen Kalender, die verschiedenen Semester um Noten einzutragen und den Schnitt zu berechnen und einen Wassertracker.

- Link zur App: https://bmld-studi-app.streamlit.app
